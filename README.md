UTEP Professor Ranking Project Setup Guide

This guide explains how to install and run the project on Windows or MacOS.

Features:
- FastAPI backend (Python)
- React + Vite frontend (Node.js)
- MongoDB Atlas Integration
- Works on Windows and MacOS
- One-click start:
    - Start.bat -> Windows
    - Start.sh -> macOS


1. Install Required Software
- Install Python 3.10+
- Install Node.js
- Install Git

2. Clone the Project
Open a terminal and run:
- git clone https://github.com/shootiettv/project.git
- cd project

3. Install Backend _ Frontend Dependencies
for Windows:  
- pip install -r requirements.txt
for MacOs:
- pip3 install -r requirements.txt

Install Node.js dependencies:
- npm install


4. Run the Project
for Windows, run in terminal:
- start.bat

for MacOS. run in terminal
- chmod +x start.sh (to give permission)
- ./start.sh (or bash start.sh if you use bash)

5. Access the App: 
Connect to 'http://localhost:5173'
